# Glitter Spa Beaches Demo Site (with images)

This is a static demo site designed for easy deployment to GitHub and Vercel.

## Included

- `index.html`
- `styles.css`
- `script.js`
- `vercel.json`
- `assets/` folder with included demo images

## Book Appointment link

All booking buttons point to:

`https://bookedin.com/book/glitter-spa-beaches/services`

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. In Vercel, click **Add New Project**.
3. Import the GitHub repository.
4. Use default settings for a static site.
5. Click **Deploy**.

## Notes

- This is a showcase/demo homepage.
- Images are bundled locally inside `/assets`.
- You can swap any image by replacing the file with the same name.
